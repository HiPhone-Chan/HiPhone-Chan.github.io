pip安装
=======

# 安装

1. 下载[ get-pip.py](https://bootstrap.pypa.io/get-pip.py)
2. 运行``` python get-pip.py ```

或者直接
  ``` curl https://bootstrap.pypa.io/get-pip.py | python ```  

# 使用
  ``` pip [un]install xxx ```

# 卸装
  ``` python get-pip.py uninstall ```
